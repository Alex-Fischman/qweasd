# QWEASD

A small game.

Controlled with:

* Q: Up
* W: Forward
* E: Down
* A: Left
* S: Backward
* D: Right
* U: Roll left
* I: Look up
* O: Roll right
* J: Look left
* K: Look down
* L: Look right
* SPACE: Shoot a laser
